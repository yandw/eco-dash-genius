

## AI+上海市工业和通信业能碳数智空间 — 门户交互界面方案

根据您提供的两张参考图，我将为现有的"节能降碳数智管理平台"新增一套**面向公众的浅色门户站点**（与现有深色后台并存）。门户采用蓝色科技风，分为首页与要闻动态等子页面。

---

### 一、整体架构

新增一个独立的 `/portal` 路由分支，与现有 `/`（深色管理后台）解耦：

```text
/portal              门户首页（参考图1）
/portal/green-mfg    绿色制造
/portal/scenarios    场景招商
/portal/news         要闻动态（参考图2）
/portal/news/:id     要闻详情
/portal/login        登录
/portal/register     注册
```

管理后台（现有全景监测等 11 个菜单）保持在 `/` 下不变。

---

### 二、视觉风格（浅色蓝调科技风）

新建独立的 `portal.css` 主题层，避免污染现有深色后台：

- 主色：科技蓝渐变 `#1E6FFF → #4DA3FF`
- 背景：白/浅灰 `#F5F8FC`
- Hero 区：蓝色科技感大图 + 半透明玻璃按钮
- 卡片：白底 + 细边框 + 悬停浮起阴影
- 字体：现代无衬线，标题加粗，副标题灰色

---

### 三、页面结构

#### 1. 门户公共框架 `PortalLayout`
- **顶部 Header**：左侧 Logo "AI+上海市工业和通信业能碳数智空间"，中间导航（首页 / 绿色制造 / 场景招商 / 要闻动态），右上角"注册 | 登录"
- **底部 Footer**：深蓝背景 + 波纹装饰，左侧"相关链接"（国家工信部 / 国家发改委 / 国家节能中心 / 上海市经信委 / 上海市发改委），右侧"上海市节能中心"联系方式，底部备案号

#### 2. 门户首页 `/portal`（参考图1）
按从上到下顺序：
- **Hero 横幅**：蓝色科技感背景 + 主标题"聚焦双碳战略，领航工业绿色转型新格局"+ 副标题"推动制造业全面绿色低碳转型"+ 双 CTA 按钮（政府管理侧 / 企业服务侧）
- **统计数据条**：5 项数字指标（10 个市级碳账户 / 17 个区级碳账户 / 80 个园区碳账户 / 130 个集团碳账户 / 1200 个企业碳账户）
- **智能体集群**：标题 + 6 个 Agent 卡片（免申即享 / 双控考核 / 节能诊断 / 绿色制造自评价 / CBAM咨询 + 1 张蓝色高亮"大模型训练数据"卡）
- **绿色制造**：标题 + 6 个图文卡片（绿色工厂、示范单位 + 4 个占位）
- **场景招商**：标题 + Tab 切换（智慧金融 / 汽车制造 / 钢铁建材 / 房地产 / 智慧出行），下方左右两个大展示区
- **业务功能**：左侧 6 个功能入口（月报填写 / 年报填写 / 限额填报 / 绿色制造申报 / XX / XX）+ 政府侧/企业侧切换；右侧蓝色 3D 立方体图形 + 围绕的功能图标
- **要闻动态**：标题 + 3 张新闻卡片（带图、标题、摘要、日期、查看详情）+ 左右切换箭头
- 末尾居中"查看更多"按钮

#### 3. 要闻动态 `/portal/news`（参考图2）
- **页面 Hero**：蓝色背景 + "要闻动态" 大标题 + 副标题"汇聚行业要闻，掌握工业绿色转型前沿资讯" + 右侧 3D NEWS 图标装饰
- **主体两栏**：
  - 左栏：分类菜单（热点动态 / 企业发布 / 通知公告），当前项蓝色高亮
  - 右栏：搜索框 + 新闻列表（图文混排卡片：左图、右标题摘要、右下角日期）

#### 4. 其他子页面
- **绿色制造**、**场景招商**、**登录**、**注册**：先建好路由与基础页面骨架（标题 + Hero + 占位内容区），保证导航连通性，具体内容后续迭代。

### 四、技术实现

| 项 | 方案 |
|---|---|
| 路由 | 在 `src/App.tsx` 新增 `/portal/*` 路由树 |
| 主题隔离 | 新建 `src/styles/portal.css`，仅在 `PortalLayout` 内通过包裹类名 `.portal-theme` 覆盖颜色变量，避免影响后台 |
| 组件目录 | `src/components/portal/`：`PortalHeader`、`PortalFooter`、`PortalLayout`、`HeroBanner`、`StatsBar`、`AgentGrid`、`GreenMfgGrid`、`ScenarioTabs`、`BusinessFunctions`（含立方体 SVG）、`NewsCarousel`、`NewsListItem` 等 |
| 页面 | `src/pages/portal/`：`PortalHome.tsx`、`PortalNews.tsx`、`PortalNewsDetail.tsx`、`PortalGreenMfg.tsx`、`PortalScenarios.tsx`、`PortalLogin.tsx`、`PortalRegister.tsx` |
| 图片资产 | Hero 背景、新闻配图、3D NEWS / 立方体装饰使用 AI 生成蓝色科技风占位图，存放 `src/assets/portal/` |
| 数据 | 新闻、Agent、绿色制造卡片均使用前端 mock 数据数组 |
| 现有后台 | 完全保留，不做修改；可在门户右上角"登录"后跳转至 `/`（现有全景监测） |
