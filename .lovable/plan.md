## 全景监测 — 多看板 + 区级画像方案

将政府端「全景监测」由单一仪表盘扩展为 **6 个二级看板**（顶部 Tab 切换），其中"首页"按参考图1完整复刻，并实现点击地图区县弹出参考图2的"区级能碳全景画像"。

---

### 一、二级菜单结构

`/gov` 下新增二级 Tab 导航，URL 同步：

| 路径 | 名称 | 内容 |
|------|------|------|
| `/gov` | **首页**（默认） | 按参考图1完整实现 |
| `/gov/decade` | 十年节能 | 十年维度的企业等级、能耗结构、强度等深度图表 |
| `/gov/dual-track` | 双控跟踪 | 能耗/碳双控月度跟踪、各区进度对比 |
| `/gov/dual-assess` | 双控考核 | 考核排行、雷达评分、年度结果 |
| `/gov/idc` | IDC监察 | PUE 分布、机柜 TOP、IDC 项目清单 |
| `/gov/equip-bench` | 设备对标 | 重点设备能效对标、超标清单 |

> 注：原文写"5个"但列了 6 项，按列出的 6 项实现。

### 二、首页看板（按参考图1，三栏布局）

```text
┌────────────────┬─────────────────────────────┬────────────────┐
│ 十年节能一眼观  │   顶部 KPI 条 (5 卡片)       │ 数据中心专项监察│
│ • 年份切换     │   536/14506.4/4326.9/       │ • 106/1.55/115.4│
│ • 企业年产值   │   21304.2/601.1             │                │
│ • 年综合能耗   ├─────────────────────────────┤ IDC 分类统计    │
│ • 仪表盘 0.273 │                             │ Tab: 综合能耗/   │
├────────────────┤   上海市重点企业地图         │ 机柜数量/PUE    │
│ 折线: 碳排放   │   (16 区轮廓 + 散点)         │ 5 档进度条      │
│ 强度 13-24年   │   中心: 上海市节能中心       ├────────────────┤
├────────────────┤                             │ 碳账户分布      │
│ 能碳双控跟踪    │                             │ 漏斗图 5 层:    │
│ • 百/千/万家%  │                             │ 企业/集团/园区/ │
│ • 行业占比环   │                             │ 区/市级         │
└────────────────┴─────────────────────────────┴────────────────┘
```

**核心组件**：
- **顶部 5 KPI 卡**：管理企业数(536/113两高)、年碳排放量、年综合能耗等价值、企业年产值、年电力消耗量；每张含"占比全市%"和"占全市工业%"双进度条
- **左侧十年一眼观**：年份 Tab(2014-2018) + 2 个统计 + 仪表盘(0-1 渐变色) + 碳排放强度折线
- **中央上海地图**：SVG 绘制简化 16 区轮廓，蓝色描边，浮动企业散点(绿/橙/红)，中心定位卡片"上海市节能中心"
- **能碳双控跟踪**：左三角金字塔(百家/千家/万家占比) + 右行业环形图(汽车制造13%)
- **右上 IDC**：3 个 KPI 横排 + Tab 切换的 5 档分级进度条
- **右下碳账户漏斗**：5 层不同颜色梯形(企业级78.13% → 市级1.30%)

### 三、区级能碳全景画像弹窗（按参考图2）

**触发**：点击中央地图任意区县轮廓 → 打开 Dialog (~960px)

**结构**：
1. 顶栏：`{区名}能碳全景画像` + 时间选择(年份下拉) + 关闭
2. **简介卡**(左半)：蓝色 3D 建筑插画背景 + 两个数据块(下属企业 100家 / 能耗等级 5000吨~50万吨)
3. **指标跟踪卡**(右半)：3 个 KPI(综合能耗(等价) 0.6 万tce / 总电能消耗 326.4 万kwh / 产值能耗 0.8 tce/万元)
4. **逐年能耗曲线**：4 个 Tab(工业产值/碳排放/综合能耗/产值能耗) + 柱状图

### 四、技术实现

| 项目 | 方案 |
|------|------|
| 路由 | `App.tsx` 新增 5 条 `/gov/*` 子路由 |
| 二级导航 | `AppLayout` 标题下方插入 `MonitoringTabs`，仅在全景监测路径显示，shadcn `Tabs` + `NavLink` |
| 页面 | `src/pages/gov/`：`Decade.tsx`、`DualTrack.tsx`、`DualAssess.tsx`、`Idc.tsx`、`EquipBench.tsx`；`Index.tsx` 重写为参考图1 |
| 首页组件 | `src/components/dashboard/overview/`：`TopKpiBar.tsx`(5卡)、`DecadeGlance.tsx`(年份+仪表盘+折线)、`ShanghaiMap.tsx`(SVG地图+点击事件)、`DualControlTrack.tsx`(金字塔+环形)、`IdcMonitor.tsx`(KPI+分级条)、`CarbonAccountFunnel.tsx`(漏斗)、`DistrictPortraitDialog.tsx`(弹窗) |
| 仪表盘/漏斗 | recharts `RadialBarChart` / 自绘 SVG 梯形 |

| 样式 | 沿用现有浅蓝科技风 token，新增渐变 KPI 卡样式 |
| 数据 | 全部前端 mock |
